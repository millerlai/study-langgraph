## 1. Model Context Protocol 概念

MCP（Model Context Protocol）是一個**開放協議**，標準化了應用程式如何向 LLM 提供工具和上下文。它就像 AI 世界的 USB-C——不論你用哪個 LLM，都能透過統一的介面使用各種工具。

```
┌────────────────────────────────────────────────────┐
│              MCP 架構概覽                            │
│                                                     │
│  ┌──────────┐    MCP 協議    ┌──────────────────┐  │
│  │ LLM 應用  │◄─────────────►│  MCP Server A    │  │
│  │ (Client)  │               │  (數學計算工具)   │  │
│  │           │               └──────────────────┘  │
│  │           │                                      │
│  │           │    MCP 協議    ┌──────────────────┐  │
│  │           │◄─────────────►│  MCP Server B    │  │
│  │           │               │  (天氣查詢工具)   │  │
│  │           │               └──────────────────┘  │
│  │           │                                      │
│  │           │    MCP 協議    ┌──────────────────┐  │
│  │           │◄─────────────►│  MCP Server C    │  │
│  │           │               │  (資料庫工具)     │  │
│  └──────────┘               └──────────────────┘  │
│                                                     │
│  關鍵概念：                                          │
│  - Server 提供 Tools / Resources / Prompts          │
│  - Client 透過標準協議連接 Server                    │
│  - 支援多種傳輸方式：stdio / HTTP                    │
└────────────────────────────────────────────────────┘
```

### MCP 三大核心功能

| 功能 | 說明 | 用途 |
|------|------|------|
| **Tools** | Server 暴露可執行的函式 | LLM 呼叫工具執行動作（搜尋、計算等） |
| **Resources** | Server 暴露資料（檔案、資料庫記錄等） | Client 讀取資料作為 LLM 上下文 |
| **Prompts** | Server 暴露可重用的 Prompt 模板 | 標準化不同場景的 Prompt |

### 傳輸方式

| 傳輸方式 | 說明 | 適用場景 |
|----------|------|---------|
| **stdio** | 透過子程序的 stdin/stdout 通訊 | 本地端工具、簡單設定 |
| **HTTP** | 透過 HTTP 請求通訊（streamable-http） | 遠端服務、生產環境 |

## 2. 建立 MCP Server

### 使用 FastMCP 建立 Server

```python
"""
建立一個簡單的 MCP Server（數學計算工具）
檔案名: math_server.py
"""
from fastmcp import FastMCP

# ============================================================
# 1. 建立 MCP Server 實例
# ============================================================
mcp = FastMCP("Math")

# ============================================================
# 2. 定義工具
# ============================================================
@mcp.tool()
def add(a: int, b: int) -> int:
    """兩個數字相加。

    Args:
        a: 第一個數字
        b: 第二個數字
    """
    return a + b

@mcp.tool()
def multiply(a: int, b: int) -> int:
    """兩個數字相乘。

    Args:
        a: 第一個數字
        b: 第二個數字
    """
    return a * b

@mcp.tool()
def divide(a: float, b: float) -> float:
    """除法計算。

    Args:
        a: 被除數
        b: 除數（不可為零）
    """
    if b == 0:
        raise ValueError("除數不能為零")
    return a / b

# ============================================================
# 3. 啟動 Server（使用 stdio 傳輸）
# ============================================================
if __name__ == "__main__":
    mcp.run(transport="stdio")
    # 也可以用 HTTP 傳輸：
    # mcp.run(transport="streamable-http")

print("MCP Math Server 定義完成")
print("工具: add, multiply, divide")
print("啟動方式: python math_server.py")
```

### 建立 HTTP 傳輸的 Server

```python
"""
建立使用 HTTP 傳輸的 MCP Server（天氣查詢工具）
檔案名: weather_server.py
"""
from fastmcp import FastMCP

mcp = FastMCP("Weather")

@mcp.tool()
async def get_weather(location: str) -> str:
    """查詢指定地點的天氣。

    Args:
        location: 城市名稱
    """
    # 模擬天氣資料
    weather_data = {
        "台北": "晴天，28°C，濕度 65%",
        "東京": "多雲，22°C，濕度 55%",
        "紐約": "雨天，15°C，濕度 80%",
    }
    return weather_data.get(location, f"{location} 的天氣資料暫時無法取得")

@mcp.tool()
async def get_forecast(location: str, days: int = 3) -> str:
    """查詢未來幾天的天氣預報。

    Args:
        location: 城市名稱
        days: 預報天數（1-7 天）
    """
    if days < 1 or days > 7:
        return "預報天數必須在 1-7 天之間"
    return f"{location} 未來 {days} 天預報: 晴時多雲，氣溫 25-30°C"

if __name__ == "__main__":
    # 使用 HTTP 傳輸，適合遠端部署
    mcp.run(transport="streamable-http")

print("MCP Weather Server 定義完成")
print("工具: get_weather, get_forecast")
print("啟動方式: python weather_server.py")
```

## 3. 在 LangGraph 中使用 MCP Server

### 安裝依賴

```bash
uv add langchain-mcp-adapters
```

### 基本用法：MultiServerMCPClient

```python
"""
使用 MultiServerMCPClient 連接多個 MCP Server
"""
import asyncio
from langchain_mcp_adapters.client import MultiServerMCPClient
from langchain.agents import create_agent

async def main():
    # ============================================================
    # 1. 設定 MCP Client（連接多個 Server）
    # ============================================================
    client = MultiServerMCPClient(
        {
            "math": {
                "transport": "stdio",            # 本地子程序通訊
                "command": "python",
                "args": ["math_server.py"],       # 指向 Server 檔案
            },
            "weather": {
                "transport": "http",              # HTTP 遠端通訊
                "url": "http://localhost:8000/mcp",
            }
        }
    )

    # ============================================================
    # 2. 取得所有工具（自動轉換為 LangChain 工具格式）
    # ============================================================
    tools = await client.get_tools()
    print(f"已載入 {len(tools)} 個工具:")
    for t in tools:
        print(f"  - {t.name}: {t.description}")

    # ============================================================
    # 3. 建立 Agent 並使用 MCP 工具
    # ============================================================
    agent = create_agent(
        "openai:gpt-4o",
        tools
    )

    # 數學問題 → 自動使用 math server 的工具
    math_result = await agent.ainvoke({
        "messages": [{"role": "user", "content": "(3 + 5) x 12 等於多少？"}]
    })
    print(f"數學結果: {math_result['messages'][-1].content}")

    # 天氣問題 → 自動使用 weather server 的工具
    weather_result = await agent.ainvoke({
        "messages": [{"role": "user", "content": "台北現在天氣如何？"}]
    })
    print(f"天氣結果: {weather_result['messages'][-1].content}")

if __name__ == "__main__":
    asyncio.run(main())
```

### HTTP 傳輸：附帶認證 Headers

```python
"""
連接需要認證的 MCP Server
"""
import asyncio
from langchain_mcp_adapters.client import MultiServerMCPClient
from langchain.agents import create_agent

async def main():
    client = MultiServerMCPClient(
        {
            "secure_api": {
                "transport": "http",
                "url": "http://api.example.com/mcp",
                "headers": {
                    "Authorization": "Bearer YOUR_API_TOKEN",
                    "X-Custom-Header": "custom-value"
                },
            }
        }
    )

    tools = await client.get_tools()
    agent = create_agent("openai:gpt-4o", tools)

    result = await agent.ainvoke({
        "messages": [{"role": "user", "content": "查詢我的帳戶資訊"}]
    })
    print(result["messages"][-1].content)

if __name__ == "__main__":
    asyncio.run(main())
```

### Stateful Sessions（有狀態的連線）

```python
"""
Stateful Sessions：保持 MCP 連線狀態
預設行為是 stateless（每次工具呼叫建立新 session）
若 Server 需要跨工具呼叫保持狀態，使用 client.session()
"""
import asyncio
from langchain_mcp_adapters.client import MultiServerMCPClient
from langchain_mcp_adapters.tools import load_mcp_tools
from langchain.agents import create_agent

async def main():
    client = MultiServerMCPClient({
        "stateful_server": {
            "transport": "http",
            "url": "http://localhost:8000/mcp",
        }
    })

    # 建立持久化的 session
    async with client.session("stateful_server") as session:
        # 在同一 session 中載入工具
        tools = await load_mcp_tools(session)

        agent = create_agent("openai:gpt-4o", tools)

        # 所有工具呼叫共用同一個 session
        # 適用於需要跨呼叫保持上下文的 Server
        result = await agent.ainvoke({
            "messages": [{"role": "user", "content": "建立一個會話並執行操作"}]
        })
        print(result["messages"][-1].content)

if __name__ == "__main__":
    asyncio.run(main())
```

## 4. 進階功能

### Tool Interceptors（工具攔截器）

Interceptors 提供 middleware 式的控制，可以修改請求、實作重試、注入上下文：

```python
"""
MCP Tool Interceptors：攔截和修改工具呼叫
"""
import asyncio
from langchain_mcp_adapters.client import MultiServerMCPClient
from langchain_mcp_adapters.interceptors import MCPToolCallRequest
from langchain.agents import create_agent

# ============================================================
# 1. 日誌攔截器
# ============================================================
async def logging_interceptor(
    request: MCPToolCallRequest,
    handler,
):
    """記錄所有工具呼叫"""
    print(f"[LOG] 呼叫工具: {request.name}, 參數: {request.args}")
    result = await handler(request)
    print(f"[LOG] 工具 {request.name} 回傳: {result}")
    return result

# ============================================================
# 2. 重試攔截器
# ============================================================
async def retry_interceptor(
    request: MCPToolCallRequest,
    handler,
):
    """失敗時自動重試（指數退避）"""
    import asyncio as aio
    max_retries = 3
    last_error = None

    for attempt in range(max_retries):
        try:
            return await handler(request)
        except Exception as e:
            last_error = e
            if attempt < max_retries - 1:
                wait = 1.0 * (2 ** attempt)
                print(f"[RETRY] 工具 {request.name} 失敗（第 {attempt+1} 次），{wait}s 後重試...")
                await aio.sleep(wait)

    raise last_error

# ============================================================
# 3. 使用攔截器
# ============================================================
async def main():
    client = MultiServerMCPClient(
        {
            "math": {
                "transport": "stdio",
                "command": "python",
                "args": ["math_server.py"],
            }
        },
        # 攔截器按洋蔥模型執行：外層 → 內層 → 工具 → 內層 → 外層
        tool_interceptors=[logging_interceptor, retry_interceptor],
    )

    tools = await client.get_tools()
    agent = create_agent("openai:gpt-4o", tools)

    result = await agent.ainvoke({
        "messages": [{"role": "user", "content": "3 + 5 等於多少？"}]
    })
    print(result["messages"][-1].content)

if __name__ == "__main__":
    asyncio.run(main())
```

### Interceptor 的洋蔥模型

```
┌─────────────────────────────────────────────────┐
│          Interceptor 執行順序（洋蔥模型）          │
│                                                  │
│  tool_interceptors=[outer, inner]                │
│                                                  │
│  ┌─────────────────────────────────────────┐     │
│  │ outer_interceptor                       │     │
│  │  print("outer: before")                 │     │
│  │  ┌───────────────────────────────┐      │     │
│  │  │ inner_interceptor             │      │     │
│  │  │  print("inner: before")       │      │     │
│  │  │  ┌─────────────────────┐      │      │     │
│  │  │  │  實際工具執行        │      │      │     │
│  │  │  └─────────────────────┘      │      │     │
│  │  │  print("inner: after")        │      │     │
│  │  └───────────────────────────────┘      │     │
│  │  print("outer: after")                  │     │
│  └─────────────────────────────────────────┘     │
│                                                  │
│  輸出順序:                                        │
│  outer: before → inner: before →                 │
│  工具執行 →                                       │
│  inner: after → outer: after                     │
└─────────────────────────────────────────────────┘
```

### Resources 與 Prompts

```python
"""
MCP Resources 與 Prompts
"""
import asyncio
from langchain_mcp_adapters.client import MultiServerMCPClient
from langchain_mcp_adapters.resources import load_mcp_resources
from langchain_mcp_adapters.prompts import load_mcp_prompt

async def main():
    client = MultiServerMCPClient({
        "my_server": {
            "transport": "http",
            "url": "http://localhost:8000/mcp",
        }
    })

    # ============================================================
    # 1. 載入 Resources（資料）
    # ============================================================
    blobs = await client.get_resources("my_server")
    for blob in blobs:
        print(f"資源 URI: {blob.metadata['uri']}")
        print(f"MIME 類型: {blob.mimetype}")
        print(f"內容: {blob.as_string()[:100]}...")

    # 也可以載入特定資源
    specific_blobs = await client.get_resources(
        "my_server",
        uris=["file:///path/to/data.json"]
    )

    # ============================================================
    # 2. 載入 Prompts（模板）
    # ============================================================
    messages = await client.get_prompt("my_server", "summarize")
    for msg in messages:
        print(f"{msg.type}: {msg.content}")

    # 帶參數的 Prompt
    messages = await client.get_prompt(
        "my_server",
        "code_review",
        arguments={"language": "python", "focus": "security"}
    )

if __name__ == "__main__":
    asyncio.run(main())
```

## 5. 重點摘要

| 主題 | 關鍵要點 |
|------|---------|
| **MCP 協議** | 開放協議，標準化 LLM 應用與工具 Server 的通訊 |
| **三大功能** | Tools（可執行函式）、Resources（資料）、Prompts（模板） |
| **傳輸方式** | stdio（本地子程序）、HTTP（streamable-http，遠端服務） |
| **FastMCP** | 快速建立 MCP Server 的 Python 框架（`@mcp.tool()` 裝飾器） |
| **langchain-mcp-adapters** | LangChain 生態系中連接 MCP Server 的官方橋接器 |
| **MultiServerMCPClient** | 同時連接多個 MCP Server，自動轉換為 LangChain 工具 |
| **Stateless vs Stateful** | 預設 stateless（每次建新 session），可用 client.session() 保持狀態 |
| **Tool Interceptors** | 洋蔥模型 middleware，可做日誌、重試、注入認證等 |
| **Headers & Auth** | HTTP 傳輸支援自訂 headers 和認證機制 |
