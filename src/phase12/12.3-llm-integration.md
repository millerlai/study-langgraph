## 1. LLM 整合概述

LangGraph 透過 LangChain 的 Chat Model 介面與各種 LLM 互動。所有 Chat Model 都實作相同的介面（`invoke`、`stream`、`bind_tools` 等），讓你可以輕鬆切換不同的 LLM 供應商。

```
┌─────────────────────────────────────────────────────┐
│              LLM 整合架構                             │
│                                                      │
│  LangGraph / LangChain 應用                           │
│      │                                               │
│      v                                               │
│  ┌───────────────────────────────────┐               │
│  │     BaseChatModel 統一介面         │               │
│  │  - invoke(messages)               │               │
│  │  - stream(messages)               │               │
│  │  - bind_tools(tools)              │               │
│  │  - with_structured_output(schema) │               │
│  └────────────┬──────────────────────┘               │
│               │                                      │
│       ┌───────┼───────┬───────┬───────┐              │
│       v       v       v       v       v              │
│  ┌────────┐┌────────┐┌────────┐┌────────┐┌────────┐ │
│  │ OpenAI ││Anthropic││ Google ││ Azure  ││ 自定義 │ │
│  │ChatGPT ││ Claude  ││ Gemini ││OpenAI  ││ LLM   │ │
│  └────────┘└────────┘└────────┘└────────┘└────────┘ │
│                                                      │
│  兩種初始化方式：                                      │
│  1. init_chat_model("provider:model")  ← 通用         │
│  2. ChatOpenAI(model="gpt-4o")         ← 特定供應商   │
└─────────────────────────────────────────────────────┘
```

## 2. 整合不同 LLM 供應商

### 方式一：init_chat_model（通用初始化）

`init_chat_model` 是最簡單的方式，只需一個字串就能初始化任何支援的模型：

```python
"""
init_chat_model：透過 model identifier string 初始化各種 LLM
格式: "provider:model_name"
"""
from langchain.chat_models import init_chat_model

# ============================================================
# 1. OpenAI
# ============================================================
openai_model = init_chat_model("openai:gpt-4o")
# 也支援簡寫（自動推斷 provider）
# openai_model = init_chat_model("gpt-4o")

# ============================================================
# 2. Anthropic Claude
# ============================================================

# ============================================================
# 3. Google Gemini
# ============================================================
gemini_model = init_chat_model("google_genai:gemini-2.0-flash")

# ============================================================
```

### 方式二：直接使用供應商套件

```python
"""
直接使用各供應商的 ChatModel 類別，可做更細緻的設定
"""
# ============================================================
# 安裝對應套件:
#   uv add langchain-openai
#   uv add langchain-anthropic
#   uv add langchain-google-genai
# ============================================================

# --- OpenAI ---
from langchain_openai import ChatOpenAI

openai_model = ChatOpenAI(
    model="gpt-4o",
    temperature=0.1,       # 控制隨機性
    max_tokens=1000,       # 最大輸出 token 數
    timeout=30,            # 請求超時秒數
    max_retries=2,         # 最大重試次數
)

# --- Anthropic ---
from langchain_anthropic import ChatAnthropic

claude_model = ChatAnthropic(
    model="claude-sonnet-4-6",
    temperature=0,
    max_tokens=2048,
)

# --- Google ---
from langchain_google_genai import ChatGoogleGenerativeAI

gemini_model = ChatGoogleGenerativeAI(
    model="gemini-2.0-flash",
    temperature=0.3,
)

# ============================================================
# 在 LangGraph 中使用
# ============================================================
from typing import TypedDict, Annotated
from langchain_core.messages import AnyMessage
from langgraph.graph import StateGraph, START, END
from langgraph.graph.message import add_messages

class ChatState(TypedDict):
    messages: Annotated[list[AnyMessage], add_messages]

def chatbot_node(state: ChatState):
    """使用指定的 LLM 進行對話"""
    # 可以替換成任何上面初始化的 model
    model = openai_model
    response = model.invoke(state["messages"])
    return {"messages": [response]}

builder = StateGraph(ChatState)
builder.add_node("chatbot", chatbot_node)
builder.add_edge(START, "chatbot")
builder.add_edge("chatbot", END)
graph = builder.compile()

print("圖已編譯，可使用任何 LLM 供應商")
# result = graph.invoke({"messages": [{"role": "user", "content": "你好"}]})
```

| 供應商 | 套件 | Model ID 格式 | 主要模型 |

| Ollama | `langchain-ollama` | `ollama:llama3` | 本地開源模型 |

### 自定義 ChatModel Wrapper

如果你想讓自定義 API 也能支援 `bind_tools`、`with_structured_output` 等 LangChain 介面：

```python
"""
實作自定義 ChatModel，完整融入 LangChain 生態系
"""
from typing import Any, Iterator, Optional, List
from langchain_core.language_models.chat_models import BaseChatModel
from langchain_core.messages import BaseMessage, AIMessage, AIMessageChunk
from langchain_core.outputs import ChatResult, ChatGeneration, ChatGenerationChunk
from langchain_core.callbacks import CallbackManagerForLLMRun

class CustomChatModel(BaseChatModel):
    """自定義 Chat Model，對接自家 API"""

    api_url: str = "http://localhost:8080/v1/chat"
    api_key: str = ""
    model_name: str = "custom-model-v1"
    temperature: float = 0.7

    @property
    def _llm_type(self) -> str:
        return "custom-chat-model"

    def _generate(
        self,
        messages: List[BaseMessage],
        stop: Optional[List[str]] = None,
        run_manager: Optional[CallbackManagerForLLMRun] = None,
        **kwargs: Any,
    ) -> ChatResult:
        """核心生成方法：呼叫你的 API"""

        # 轉換 messages 為你的 API 格式
        api_messages = []
        for msg in messages:
            role = "user" if msg.type == "human" else "assistant"
            api_messages.append({"role": role, "content": msg.content})

        # 模擬 API 呼叫（實際應用換成 requests.post）
        # response = requests.post(
        #     self.api_url,
        #     json={"messages": api_messages, "temperature": self.temperature},
        #     headers={"Authorization": f"Bearer {self.api_key}"},
        # )
        # content = response.json()["choices"][0]["message"]["content"]

        content = f"[{self.model_name}] 收到 {len(messages)} 則訊息，溫度={self.temperature}"

        return ChatResult(
            generations=[
                ChatGeneration(message=AIMessage(content=content))
            ]
        )

    def _stream(
        self,
        messages: List[BaseMessage],
        stop: Optional[List[str]] = None,
        run_manager: Optional[CallbackManagerForLLMRun] = None,
        **kwargs: Any,
    ) -> Iterator[ChatGenerationChunk]:
        """串流生成方法"""
        # 模擬串流回應
        content = f"[{self.model_name}] 串流回應"
        for char in content:
            yield ChatGenerationChunk(
                message=AIMessageChunk(content=char)
            )

# ============================================================
# 使用自定義 ChatModel
# ============================================================
model = CustomChatModel(
    api_url="http://my-api.example.com/v1/chat",
    api_key="sk-xxx",
    model_name="my-llm-v2",
    temperature=0.3,
)

# 跟標準 LangChain ChatModel 一樣使用
response = model.invoke([{"role": "user", "content": "你好"}])
print(f"回應: {response.content}")

# 串流
for chunk in model.stream([{"role": "user", "content": "你好"}]):
    print(chunk.content, end="")
print()

# 在 LangGraph 中使用
from typing import TypedDict, Annotated
from langchain_core.messages import AnyMessage
from langgraph.graph import StateGraph, START, END
from langgraph.graph.message import add_messages

class State(TypedDict):
    messages: Annotated[list[AnyMessage], add_messages]

def agent_node(state: State):
    response = model.invoke(state["messages"])
    return {"messages": [response]}

builder = StateGraph(State)
builder.add_node("agent", agent_node)
builder.add_edge(START, "agent")
builder.add_edge("agent", END)
graph = builder.compile()

result = graph.invoke({"messages": [{"role": "user", "content": "你好"}]})
print(f"圖輸出: {result['messages'][-1].content}")
# 輸出:
# 回應: [my-llm-v2] 收到 1 則訊息，溫度=0.3
# [my-llm-v2] 串流回應
# 圖輸出: [my-llm-v2] 收到 1 則訊息，溫度=0.3
```

### 方式二：Middleware 動態模型選擇

```python
"""
使用 Middleware 在 runtime 根據條件動態選擇 LLM
"""
from langchain_openai import ChatOpenAI
from langchain.agents import create_agent
from langchain.agents.middleware import wrap_model_call, ModelRequest, ModelResponse
from langchain.tools import tool

# ============================================================
# 1. 預備多個模型
# ============================================================
fast_model = ChatOpenAI(model="gpt-4o-mini", temperature=0)    # 快速便宜
smart_model = ChatOpenAI(model="gpt-4o", temperature=0)        # 聰明但較貴

# ============================================================
# 2. 建立動態選擇 Middleware
# ============================================================
@wrap_model_call
def dynamic_model_selection(request: ModelRequest, handler) -> ModelResponse:
    """根據對話複雜度動態選擇模型"""
    messages = request.state["messages"]
    message_count = len(messages)

    # 策略：長對話用聰明模型，短對話用快速模型
    if message_count > 10:
        selected_model = smart_model
        reason = "長對話 → 使用 GPT-4o"
    else:
        selected_model = fast_model
        reason = "短對話 → 使用 GPT-4o-mini"

    print(f"[MODEL SELECT] {reason} (訊息數: {message_count})")
    return handler(request.override(model=selected_model))

# ============================================================
# 3. 建立 Agent
# ============================================================
@tool
def search(query: str) -> str:
    """搜尋資訊。"""
    return f"搜尋結果: {query}"
```

### 方式三：透過 Config 在 Node 中切換

```python
"""
在 LangGraph 節點中透過 config 動態切換 LLM
"""
from typing import TypedDict, Annotated
from langchain_core.messages import AnyMessage
from langchain_core.runnables import RunnableConfig
from langchain.chat_models import init_chat_model
from langgraph.graph import StateGraph, START, END
from langgraph.graph.message import add_messages

# ============================================================
# 1. 定義 State
# ============================================================
class ChatState(TypedDict):
    messages: Annotated[list[AnyMessage], add_messages]

# ============================================================
# 2. 預建模型字典
# ============================================================
MODELS = {
    "openai": init_chat_model("openai:gpt-4o"),
    "anthropic": init_chat_model("anthropic:claude-sonnet-4-6"),
    "fast": init_chat_model("openai:gpt-4o-mini"),
}

# ============================================================
# 3. 節點：根據 config 選擇模型
# ============================================================
def llm_node(state: ChatState, config: RunnableConfig) -> dict:
    """根據 config 中的 model_choice 選擇 LLM"""
    model_choice = config.get("configurable", {}).get("model_choice", "fast")
    model = MODELS.get(model_choice, MODELS["fast"])

    print(f"[使用模型] {model_choice}")
    response = model.invoke(state["messages"])
    return {"messages": [response]}

# ============================================================
# 4. 建立圖
# ============================================================
builder = StateGraph(ChatState)
builder.add_node("llm", llm_node)
builder.add_edge(START, "llm")
builder.add_edge("llm", END)
graph = builder.compile()

# ============================================================
# 5. 執行：透過 config 切換模型
# ============================================================

# 使用 OpenAI
# result = graph.invoke(
#     {"messages": [{"role": "user", "content": "你好"}]},
#     config={"configurable": {"model_choice": "openai"}}
# )

# 使用 Anthropic
# result = graph.invoke(
#     {"messages": [{"role": "user", "content": "你好"}]},
#     config={"configurable": {"model_choice": "anthropic"}}
# )

# 使用快速模型（預設）
# result = graph.invoke(
#     {"messages": [{"role": "user", "content": "你好"}]},
#     config={"configurable": {"model_choice": "fast"}}
# )

print("動態 LLM 切換圖已建立")
print("透過 config['configurable']['model_choice'] 切換:")
print("  'openai'    → GPT-4o")
print("  'anthropic' → Claude Sonnet")
print("  'fast'      → GPT-4o-mini（預設）")
```

## 5. 重點摘要

| 主題 | 關鍵要點 |
|------|---------|
| **init_chat_model** | 通用初始化函式，格式 `"provider:model"`，支援自動推斷 |
| **供應商套件** | 各供應商有獨立套件（langchain-openai、langchain-anthropic 等） |
| **統一介面** | 所有 ChatModel 共用 invoke/stream/bind_tools 介面 |
| **直接使用 API** | 可在 Task/Node 中直接呼叫任何 API，不必透過 LangChain |
| **自定義 ChatModel** | 繼承 BaseChatModel，實作 `_generate` 和 `_stream` 方法 |
| **Configurable Fields** | 透過 configurable_fields 讓模型參數可在 runtime 變更 |
| **Middleware 選擇** | 用 @wrap_model_call 根據 state/context 動態選擇模型 |
| **Config + Node** | 在 Node 中讀取 RunnableConfig 手動切換模型 |
| **成本優化** | 簡單任務用便宜模型、複雜任務用強力模型 |
